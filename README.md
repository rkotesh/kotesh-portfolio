# Sankula Koteswara Rao Portfolio

A modern, responsive personal portfolio website to showcase projects, skills, education, achievements, and contact information.

Live site: https://rkotesh.github.io/kotesh-portfolio/

## Highlights

- Hero section with signature-style branding and resume CTA
- About, skills, education, achievements, and contact sections
- 12 featured projects grouped by type
- Individual project thumbnails for each card
- Click-to-open project details modal
- Rolla AI assistant with quick actions and portfolio-aware responses
- Responsive layout with a premium Slate + Teal visual direction
